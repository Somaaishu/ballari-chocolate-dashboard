Ballari Chocolate Dashboard

Ballari Chocolate Dashboard is a React + Vite web application for analyzing chocolate sales and performance data through an interactive and responsive dashboard. It uses Recharts for charts, PapaParse for CSV handling, and Tailwind CSS for styling.

Repository
https://github.com/Somaaishu/ballari-chocolate-dashboard

Live Demo
https://project-sigma-rouge-95.vercel.app/

Tech Stack
HTML, CSS, JavaScript, React

Getting Started
Install dependencies:
```bash
npm install
Run in development mode:

npm run dev
Create production build:

npm run build
Preview production build locally:

npm run preview
Deployment (GitHub Pages)
Deploy command:

npm run deploy
After running deploy, configure GitHub Pages:

Open repository Settings
Go to Pages
Source: Deploy from a branch
Branch: gh-pages
Folder: / (root)
Expected hosted URL: https://somaaishu.github.io/ballari-chocolate-dashboard/

Available Scripts
npm run dev - Start Vite development server
npm run build - Build app for production
npm run preview - Preview production build
npm run deploy - Deploy dist to GitHub Pages
